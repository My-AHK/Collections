

Core Measurement Tools:
    • Interactive Distance & Angle Measurement: Measures both pixel distance and the angle between any two points on the screen.
    • Dual-Mode Operation: Horizontal ruler for width measurement, vertical ruler for height measurement.
    • Click-to-Measure Workflow: Two-point measurement system activated via keyboard shortcuts.
    • Real-Time Measurement Display: Shows calculated results directly on the ruler interface.
    • Multi-Unit Conversion: Automatically converts pixel distance to inches, centimeters, and millimeters.
    • Angle Calculation: Computes the angle of the measurement line relative to the positive x-axis (0 right, positive up, -180 to 180 range).
    • Configurable PPI Setting: Adjustable pixels-per-inch value for accurate real-world unit conversion (default: 96 PPI).
    • Always-On-Top Window: Ruler remains visible above other windows for convenient measurement reference.

Precise Ruler Display:
    • Accurate Tick Mark System: Ruler ticks at 10-pixel intervals with major ticks at 50-pixel increments.
    • Bidirectional Tick Marks: Ticks appear on both top/bottom (horizontal) or left/right (vertical) edges.
    • Numeric Labeling: Major ticks labeled with pixel position values at 50-pixel intervals.
    • Configurable Ruler Length: Adjustable ruler scale (default: 800 pixels).
    • Customizable Ruler Thickness: Adjustable bar width (default: 70 pixels).
    • Outline Border: 1-pixel outline around the main ruler bar for visual clarity.

Measurement Workflow System:
    • Four-State Mode Tracking:
        ⮞ Idle  Ready state with instructional text
        ⮞ Awaiting Start  Activated mode waiting for first point
        ⮞ Live Preview  First point set, shows live angle/distance tooltip at cursor
        ⮞ Awaiting Stop  (Legacy state replaced by Live Preview) First point set, waiting for second point
    • Ctrl+Alt+M Activation: Toggles measurement mode on.
    • Ctrl+Alt+LClick Point Setting: Sets start and end points for distance/angle calculation.
    • Live Angle Tooltip: When the start point is set, a tooltip follows the cursor showing real-time angle and distance until the end point is placed.
    • Dynamic Instructional Text: Display updates to guide user through the measurement process.
    • Automatic Mode Reset: Measurement mode deactivates after completion.
    • Escape Key Clear: Cancels active measurement, clears all points, resets to idle state, and hides any active tooltip.

Distance Calculation Engine:
    • Pythagorean Distance: Uses (dx + dy) for accurate diagonal measurement.
    • Angle Calculation: Uses ATan2 with a custom function to provide continuous angle feedback from -180 to 180.
    • Auto Unit Conversion:
        ⮞ Pixels  Primary measurement unit
        ⮞ Inches  Pixel distance  PPI
        ⮞ Centimeters  Inch value  2.54
        ⮞ Millimeters  Centimeter value  10
    • Formatted Output Display: Results shown with appropriate decimal precision per unit, including the calculated angle in degrees.

Clipboard Integration:
    • Automatic Copy on Measure: Measurement result and angle are automatically copied to the clipboard upon completion.
    • Manual Copy Shortcut (Ctrl+Alt+C): Copies current measurement text to the clipboard.
    • Copy Validation: Prevents copy attempts when no measurement exists.
    • Tooltip Confirmation: Provides visual feedback when a measurement is copied or if the action is unavailable.

Ruler Lifecycle & States:
    • Launch  Tray icon appears, ruler GUI built and shown.
    • Idle  Ruler displays with instructional text.
    • Measure Mode Active  User activates via Ctrl+Alt+M.
    • Point 1 Set (Live Preview)  Start coordinates captured, live angle/distance tooltip follows cursor.
    • Point 2 Set  Final distance and angle are calculated, results are displayed and copied.
    • Orientation Toggle  GUI destroyed and rebuilt in new orientation; measurement mode is reset.
    • Minimize  Window hidden, tray icon updates.
    • Restore  Window reshown, tray icon updates.
    • Reload  Script auto-reloads on file modification.
    • Exit  Sound plays, application terminates.

Error Handling & Safeguards:
    • Mode Validation: Ctrl+Alt+LClick is ignored if measurement mode is not active.
    • Measurement State Tracking: Prevents conflicting measurement attempts.
    • Copy Protection: Validates that a measurement exists before copying to the clipboard.
    • Hidden Window Detection: RestoreRulerWindow checks window state before toggling.
    • File Time Initialization: Prevents an empty modification time from triggering a reload.
    • Color Fallback: Unrecognized color names default to White (FFFFFF).
    • Tooltip Cleanup: The live preview tooltip is explicitly cleared when a measurement is finalized or canceled.
