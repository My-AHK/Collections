
Core Conversion Functionality:
    • 18 distinct conversion categories:
        ▸ Length, Weight, Volume, Area, Temperature
        ▸ Speed, Time, Digital Storage, Currency
        ▸ Pressure, Energy, Power, Fuel Consumption
        ▸ Data Transfer, Angle, Torque, Force, Acceleration
   • Real-time automatic conversion as values are typed
   • Bidirectional unit swap with single click (↔ button)
   • Full-precision result display (up to 15 decimal places, trailing zeros removed)
   • Scientific notation for very large or small results
   • Default starting tab configurable via variable

        ▸ Individual tab header colors (19 category-specific colors)
        ▸ Input text, result text, full precision text per category
        ▸ Intro tab link color, return arrow color
   • Predefined color palette with 40+ named colors:
        ▸ Grayscales, Reds, Pinks, Oranges, Yellows, Greens, Cyans, Blues, Violets, Magenta
   • Full 6-digit hex code support (with or without 0x prefix)
   • Window transparency control (0-255) via hotkey toggle

User Interface:

Conversion Logic Details:
   • Base-unit conversion architecture (meters, grams, liters, etc.)
   • Temperature uses formula-based conversion (non-linear)
   • Fuel consumption uses special MPG/L·100km reciprocal formulas
   • Digital Storage uses 1024-based binary prefixes
   • Time uses 365.25-day years for leap year averaging
   • Speed includes Mach unit (343 m/s standard)
   • Acceleration supports g-force, Gal, and km/h/s units
   • Torque uses dyne-centimeters for small-scale measurements
