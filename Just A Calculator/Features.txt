
Core Calculator Functionality:
    • Full Expression Evaluation: Supports complete mathematical expressions including nested parentheses and exponentiation.
    • Custom Math Parser: Independently processes expressions without relying on external evaluation functions.
    • Operator Precedence Handling: Correct order of operations (power → multiply/divide → add/subtract).
    • Error Detection: Identifies invalid expressions and division-by-zero scenarios with user feedback.
    • Formatted Results: Automatically removes trailing zeros and rounds results cleanly.

Input & Interaction:
    • Dual Input Methods: Fully functional via both GUI buttons and keyboard input.
    • Numpad Support: Seamless integration with numpad keys for rapid entry.
    • Smart Operator Handling: Prevents duplicate operators and auto-formats spacing.
    • Backspace & Clear: Supports character removal and full display reset.
    • Parentheses Support: Allows complex nested calculations.

Result Handling & Clipboard:
    • Live Result Display: Shows full equation and result after calculation.
    • Click-to-Copy:
        ⮞ Click = Copies result only.
        ⮞ Ctrl + Click = Copies full equation.
    • Visual Feedback: Result text briefly changes color when copied.
    • Error Reset Timer: Automatically clears error state after delay.

Calculation History System:
    • Persistent History Storage: Saves calculations to INI file.
    • History Limit: Stores up to 20 recent entries.
    • History Window:
        ⮞ Displays all past calculations in list format.
        ⮞ “Use” button loads selected expression back into display.
    • Clear History Option: Removes all stored entries with confirmation.
    • Session Persistence: History is restored on script restart.

Memory Functions:
    • MC (Memory Clear): Resets stored memory value.
    • MR (Memory Recall): Loads memory value into display.
    • M+ / M-: Adds or subtracts current value to/from memory.
    • MS (Memory Store): Saves current value directly to memory.
    • Memory Indicator: Visual cue when memory is active (non-zero).

Mathematical Functions:
    • Square Root (sqrt): Calculates √ of current value.
    • Percentage (%): Converts value into percentage form.
    • Plus/Minus (±): Toggles sign of current number.
    • Power Operator (^): Supports exponent calculations.

User Interface Layout:
    • Structured Grid Layout: Organized button placement with consistent spacing.

    • Result Text Line: Separate line for equation + result.
    • Function Rows: Clearly separated advanced function buttons.
