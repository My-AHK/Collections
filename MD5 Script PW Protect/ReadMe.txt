
	* Instructions for Use *

1. Generate Hashes:
    ▹ Run MD5 Hasher.ahk.
    ▹ Enter the desired username or password into the input box.
    ▹ Copy the resulting MD5 hash from the clipboard.

2. Embed Hashes:
    ▹ Open MD5 Script Protect.ahk.
    ▹ Replace User and Pass variables with the copied MD5 hashes for your username and password.

3. Secure Your Script:
    ▹ Save and use MD5 Script Protect.ahk to restrict access to the script or functionality requiring credentials.
