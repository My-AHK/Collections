

NEW FEATURES:
    ★ Replace Mode Added
        • New ReplaceRadio option
        • Allows complete filename overwrite (instead of only prepend/append)
    ★ Revert (Undo) System
        • Creates automatic rename backup in INI
        • RevertFiles function restores original filenames
        • Deletes backup after successful revert
        • Prevents revert if no backup exists
    ★ Per-Extension Sequential Numbering
        • Numbering restarts for each file extension group
        • Example:
            ▸ .jpg → 001, 002, 003
            ▸ .png → 001, 002
        • More logical grouping than global numbering

    ★ Sorted File Display
        • Files sorted by:
            ▸ Extension (alphabetical)
            ▸ Then filename (alphabetical)
        • Custom sort function: SortFilesByExtensionAndName()
    ★ Automatic Checkbox State Monitoring
        • Timer-based CheckCheckboxChanges
        • Preview updates automatically when checkboxes change
        • No manual preview refresh required
    ★ INI-Based Persistence
        • Saves last used folder
        • Automatically reloads it on next script launch
        • Falls back to default if folder no longer exists


BACKUP & DATA HANDLING IMPROVEMENTS

    ★ Hidden INI file
